# Resumen de requisitos

- Catálogo público.
- Sin autenticación.
- Funcionamiento offline.
- HTML navegable.
- Uso en computador, celular y tablet.
- Colores institucionales: Green #71b956, Turquoise #2cab87, Black #000000.
- Tipografías solicitadas: Nunito Sans y Montserrat.
- Contenido: nombre común, nombre científico, fotografías, descripción, propagación, zona geográfica, estado de conservación, cuidados, usos ecológicos, mapas, PDF y fichas descargables.
- Actualización por soporte IT y profesional de restauración.
